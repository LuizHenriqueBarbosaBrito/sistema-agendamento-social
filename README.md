# 🗓️ Sistema de Agendamento Social

## 📌 Sobre o Projeto
O **Sistema de Agendamento Social** foi desenvolvido como parte da disciplina **Atividade Extensionista** do curso de Análise e Desenvolvimento de Sistemas (UNINTER). 

O objetivo principal é oferecer uma plataforma funcional para organizar atendimentos comunitários, facilitando o cadastro de usuários e beneficiários, além de gerenciar os horários de agendamentos de forma eficiente e simples.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Java
* **Framework:** Spring Boot (Spring Data JPA, Spring Web)
* **Banco de Dados:** MySQL / H2 Database
* **Gerenciador de Dependências:** Maven
* **Interface:** (Se você usou HTML/Thymeleaf ou apenas API, mencione aqui)

## 🛠️ Funcionalidades
- [x] **Cadastro de Usuários:** Registro de operadores do sistema.
- [x] **Gestão de Beneficiários:** Cadastro de pessoas que buscam atendimento comunitário.
- [x] **Agendamento:** Sistema para marcar, listar e gerenciar horários de atendimento.
- [x] **Impacto Social:** Ferramenta pensada para facilitar o trabalho de ONGs e centros comunitários.
